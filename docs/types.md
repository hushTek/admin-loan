## Loan Types

*   ### SALARY ADVANCE 

        Interest  = 10%
        Duration = 1 Month
        e.g 100_000 -> (100_000 + 10_000(10%)) -> 110_000

*   ### DOUBLE MONTH LOAN

        Interest = 16% 
        Duration = 2 Month
        e.g 100_000 -> (100_000 + 16_000(16%)) -> 116_000 
        Person will pay 116_000 / Duration ( 2 Month) -> 58_000

*   ### QUARTERLY ADVANCE LOAN 

        Interest = 23%
        Duration = 3 Month
        e.g 100_000 -> (100_000 + 23_000(23%)) -> 123_000
        Person will pay 123_000 / Duration ( 3 Month) -> 41_000
